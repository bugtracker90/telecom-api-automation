This project demonstrates API testing and automation for the Telecom Contact List application.

## Tools Used
- Postman
- RestAssured
- Java
- TestNG
- Maven

## API Testing Flow
1. Add User
2. Get User Profile
3. Update User
4. Login User
5. Add Contact
6. Get Contact List
7. Get Contact
8. Update Contact
9. Logout User

## Automation
The APIs are automated using RestAssured in Eclipse with TestNG framework.
